- Q6：递归中，如果某次递归需要调用自身两次，可以把第一调用的结果存下来，避免再次计算的开销！

  ```C++
  // good version
  string starString(int n)
  {
      if (n < 0)
          throw "Invalid input.";
      else if (n == 0)
          return "*";
      else
      {
          string star = starString(n - 1);
          return star + star;
      }
  }
  // bad version
  string starString(int n)
  {
      if (n < 0)
          throw "Invalid input.";
      else if (n == 0)
          return "*";
      else
      {
          return starString(n - 1) + starString(n - 1);
      }
  }
  ```

  分析调用次数可以发现，第一个版本调用次数是随n线性变化的，而第二个版本是指数变化
